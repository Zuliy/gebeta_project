# Ethio Gebeta *999# USSD Simulator

## Project Description
This is a shell script program that simulates the Ethio Telecom *999# USSD service. Users can browse and purchase data packages, voice packages, and combo offers through a simple menu system.

## Features
- Internet/Data packages (Daily, Weekly, Monthly)
- Voice call packages (Daily, Weekly, Monthly)
- Social media packages (WhatsApp, Facebook, TikTok)
- Combo packages (Voice + Data)
- Telebirr special offers
- Unlimited packages
- Order confirmation system

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run: `./main.sh`

## Requirements
- Linux/Unix environment (Ubuntu, WSL, Git Bash)
- Bash shell
